# Weather App

A responsive weather application built using HTML, CSS, and JavaScript that fetches real-time weather information using the OpenWeatherMap API.

## Features

- Search weather by city name
- Display temperature
- Show humidity and wind speed
- Dynamic weather icons
- Error handling for invalid city names
- Responsive UI

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
